# 🎮 Minicraft - 單人無限遊戲

一個用 JavaScript 開發的 Minecraft 風格遊戲，包含無限地圖、多種地形和背包系統。

## 🎯 功能特性

- **無限地圖生成** - 使用 Perlin 噪聲算法動態生成
- **多種地形** - 草地、石頭、泥土、沙子、水等
- **玩家系統** - 移動、跳躍、重力物理
- **背包系統** - 可收集物品和資源管理
- **HUD 顯示** - 實時坐標、健康值顯示

## 🕹️ 控制方式

- **W/A/S/D** - 移動
- **空格鍵** - 跳躍
- **E** - 打開/關閉背包
- **滑鼠** - 看向（待實現）

## 📁 項目結構

```
minicraft/
├── index.html       # 主頁面
├── style.css        # 樣式表
├── README.md        # 說明文件
└── js/
    ├── main.js      # 遊戲主程式
    ├── world.js     # 世界生成
    ├── player.js    # 玩家系統
    ├── inventory.js # 背包系統
    └── terrain.js   # 地形定義
```

## 🚀 如何開始

1. 克隆倉庫
```bash
git clone https://github.com/B13505015/minicraft.git
```

2. 打開 `index.html` 在瀏覽器中

3. 開始遊玩！
